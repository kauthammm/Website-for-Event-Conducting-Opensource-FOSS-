# Military Hackathon Website Requirements Document

## 1. Website Name
Military Hackathon Command Center

## 2. Website Description
An intense, battlefield-themed frontend website designed for hosting a military hackathon event. The site immerses participants in a realistic combat environment with gritty visuals, bullet-point formatting, and tactical imagery. Features an interactive hover military character in the foreground and a game-style story narration panel mounted on the side. This pure frontend application delivers static content with a visceral, high-stakes atmosphere that captures the intensity of military operations, with all text rendered in striking neon orange for maximum visibility.

\n## 5. Design Style
\n• **Color Scheme**: Gritty battlefield palette with desert tan (#C19A6B), gunmetal gray (#2C3539), blood red (#8B0000) for alerts, smoke black (#0D0D0D) for backgrounds, brass bullet gold (#B5A642) for accents, **neon orange (#FF6600) for all primary text, headings, and interactive elements**
• **Visual Details**: Distressed textures with scratches and wear marks, bullet hole effects on borders, spent shell casing icons for bullet points, razor wire dividers between sections, dog tag styled badges, blood splatter accents (subtle, artistic), weathered metal textures on buttons and panels, **interactive hover military character (WhatsApp Image 2025-12-20 at 1.28.08 PM.jpeg) with smooth parallax movement effect responding to cursor position**
• **Layout**: Tactical briefing document structure with all content in bullet-point format, command dossier aesthetic with stamped classifications, rugged typography using military stencil fonts (headers) and typewriter-style fonts (body text), asymmetric grid suggesting battlefield chaos while maintaining readability, **side-mounted story narration panel (left or right edge) with semi-transparent dark background (#0D0D0D with85% opacity) and auto-scrolling game-style narrative text in neon orange**, full-screen background image (image.png) covering entire viewport\n• **Interactive Elements**: Bullet casings as list markers, crosshair hover effects, shell ejection animations on button clicks, flickering light effects suggesting combat conditions, **military character that follows mouse movement with smooth parallax effect creating depth and immersion**
• **Typography**: All text rendered in neon orange (#FF6600) for high visibility against dark battlefield backgrounds, creating cyberpunk-military fusion aesthetic with strong contrast and readability\n• **Story Narration Panel**: Fixed position panel mounted on side of screen, semi-transparent dark background, auto-scrolling text in typewriter-style font, neon orange text color, cinematic game-style narrative format, content provided by user
